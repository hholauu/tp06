Objetivos:

  • Familiarizar a los estudiantes con las características y sintaxis de CSS.
  • Crear estilos básicos para sus páginas web.
  • Ayudar a los estudiantes a entender cómo se relacionan los estilos con el HTML.
  • Ayudar a los estudiantes a entender la anatomía de CSS, incluyendo selectores, propiedades y valores.
La importancia de aprender CSS
1. CSS es una forma de hacer que las cosas se vean bonitas en la pantalla de una computadora.
2. Ayuda a cambiar los colores, las fuentes y los tamaños de cosas como palabras e imágenes.
4. CSS se asegura de que todo se vea bien y organizado.
5. Sin CSS, todo se vería simple y aburrido.

1.1- ¿Qué significa CSS?

CSS son las siglas de “Cascading Style Sheets” (hojas de estilo en cascada). CSS es un lenguaje que maneja el diseño y presentación de las páginas web (HTML o XML).
